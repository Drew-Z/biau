# UI Review

请从产品官网和展示系统角度审查 UI。

重点：

1. 首页是否大气、是否有明确首屏、项目分区和案例矩阵。
2. 项目页是否分为 AI 应用、全栈开发、游戏项目，并支持清晰选择。
3. 案例页是否以业务问题、方案、结果、证据组织。
4. 博客页是否与整体风格统一，同时有博客站的阅读感。
5. 详情页是否与列表页有明显区分，不要点击后像没跳转。
6. 字体层级、间距、对齐、卡片密度、浅暗主题是否一致。
7. 移动端是否无文字溢出、无元素重叠。
8. 如可运行本地服务，请结合 Playwright MCP 或浏览器截图检查。

输出问题清单时按严重程度排序，并给出可执行修改建议。
